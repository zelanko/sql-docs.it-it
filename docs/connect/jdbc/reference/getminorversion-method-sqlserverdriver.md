# <a name="getminorversion-method-sqlserverdriver"></a>Metodo getMinorVersion (SQLServerDriver)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Restituisce il numero di versione secondaria di [!INCLUDE[jdbcNoVersion](../../../includes/jdbcnoversion_md.md)].  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public int getMinorVersion()  
```  
  
## <a name="return-value"></a>Valore restituito  
 Valore **int** contenente il numero di versione secondaria del driver JDBC.  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo getMinorVersion viene specificato dal metodo getMinorVersion nell'interfaccia java.sql.Driver.
