# <a name="getlabel-method-sqlserversavepoint"></a>Metodo getLabel (SQLServerSavepoint)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Ottiene il nome dell'etichetta del punto di salvataggio.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public java.lang.String getLabel()  
```  
  
## <a name="return-value"></a>Valore restituito  
 Valore **String** contenente il nome dell'etichetta del punto di salvataggio.
