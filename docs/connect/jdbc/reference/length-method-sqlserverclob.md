# <a name="length-method-sqlserverclob"></a>Metodo length (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Restituisce il numero di caratteri nell'oggetto CLOB.  
  
## <a name="syntax"></a>Sintassi  
  
```  
  
public long length()  
```  
  
## <a name="return-value"></a>Valore restituito  
 Lunghezza dell'oggetto CLOB in caratteri numerici.  
  
## <a name="exceptions"></a>Eccezioni  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Osservazioni  
 Questo metodo length viene specificato dal metodo length nell'interfaccia java.sql.Clob.
