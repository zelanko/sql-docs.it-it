# <a name="iswebserviceenabled-property-wmi-msreportserver_configurationsetting"></a>Proprietà IsWebServiceEnabled (MSReportServer_ConfigurationSetting WMI)
  Indica se il servizio Web ReportServer è abilitato. Di sola lettura.  
  
## <a name="syntax"></a>Sintassi  
  
```vb  
Public Dim IsWebServiceEnabled As Boolean  
```  
  
```csharp  
public boolean IsWebServiceEnabled;  
```  
  
## <a name="property-values"></a>Valori della proprietà  
 Valore `Boolean` di sola lettura. Un valore `true` indica che il servizio Web ReportServer è abilitato.
