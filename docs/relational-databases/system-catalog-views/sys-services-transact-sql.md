[!INCLUDE [SQL Server](../../includes/applies-to-version/sqlserver.md)]

|Nome colonna|Tipo di dati|Descrizione|  
|-----------------|---------------|-----------------|

|**service_id**|**int**|Identificatore del servizio. Non ammette i valori Null.|
